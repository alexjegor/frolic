# IoT & Software Architecture

## System Architecture

Typical IoT setup:

ESP32
→ sensors
→ actuator control
→ WiFi telemetry
→ dashboard

Sensors commonly used:

- pH sensor
- EC/TDS sensor
- water level sensor
- temperature sensor
- humidity sensor

Automation functions:

- pump scheduling
- lighting cycles
- environmental monitoring
- alerts
