# Hydroponics + IoT + 3D Printing

Indoor hydroponic system with ESP32-based monitoring, automated irrigation, and 3D printed structural components.

## Goals

- Efficient indoor cultivation without soil
- IoT monitoring: temperature, humidity, water level, pH, EC/TDS
- Automated pump scheduling, lighting, alerts
- Simple plumbing with easy maintenance
- 3D printed components where practical

Research phase — method and architecture not yet chosen.
